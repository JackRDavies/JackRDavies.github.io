Networking

One of the 2nd most important topics of this course is being able to understand the computer networking topologies, which is a big deal when it comes to todays Business IT sector as it allows computers to interact with one another. Throughout the course we were informed of networking subjets, routers, switches, firewalls, routers and other fundamentals such as types of cables and connectors, e.g. CAT5, RJ45 etc.

One of the tasks that we were assigned during this unit was to make a functional Windows-Based network that allowed computers to access a shared resource with different user accounts and group policies. Of course, for me this was an easy subject as I have networking experience from my Robotics project and Webserver project.

IT networking and security is fundamental in this area of computing and is a must for any Administrator or website producer as it is the magic that allows a computer to interconnect with one another, allowing people to access websites and information.

The task I was assigned was fairly straight forward as Microsoft provide both GUI and CLI methods of creating and managing a computer network. In the scenario I was provided, I had to create a star topology network which required the following devices:
  • Ethernet cable with RJ45 connector
  • Router
  • 3 computers/laptops with RJ45 Ethernet ports.
